# @alicloud/xconsole-theme

- `index.scss` 是导出全量 console-components 样式。已编译成 css 放在`dist/index.css`
- `index-noreset.scss` 是导出全量 console-components 样式(不包含 reset.css)，已编译成 css 放在`dist/index-noreset.css`
- `design-tokens/` 是设计师在 fusion 平台上配置好的设计参数。如果你想使用设计师配置的变量，用它
- `style-overwrite/` 是 console-components 对 fusion 组件样式的覆盖，用户一般不需要
- `styles/` 下面是一些工具样式，用户一般不需要

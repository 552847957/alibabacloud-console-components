# Timeline 时间轴

使用场景

- 当有一系列信息需要从上至下按时间排列时。
- 需要有一条时间轴进行视觉上的串联。


## 基本用法

tilte 属性设置时间轴的文字描述，time 设置时间，state 设置节点状态。

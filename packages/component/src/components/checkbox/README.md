# Checkbox 复选框

使用场景：复选框允许用户从一个数据集中选择多个选项。

## 基本

Checkbox 组件使用，分 label 和无 label 两种情况。defaultChecked 属性默认选中，checked 属性选中，disabled 属性不可操作，defaultIndeterminate 属性设置半选状态。

# SplitButton 分隔按钮

使用指南

SplitButton 同样由 Button 和 Menu 组成，其在触发区域上可以分为两部分，其中 Icon 部分作为弹层菜单的触发区域。 菜单只有在鼠标悬浮或点击右侧图标区域的时候才会展示。

## 基本

最简单的用法。支持 Button 的 type, size, component, ghost 等属性透传。
